## Instalação

Para rodar o **Aerocode** localmente, siga os passos abaixo:

1. **Clone o repositório**
   ```bash
   git clone https://github.com/CauaCurisno1446/AV3.git
   cd AV3
   ```

2. **Instale as dependências**
   ```bash
   npm install
   ```

3. **Execute o sistema**
   ```bash
   npm run dev
   ```

4. Acesse no navegador:
   ```
   http://localhost:5173/
   ```
